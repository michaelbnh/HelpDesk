# 🛠️ Sistema de Helpdesk

Sistema completo de gerenciamento de chamados de suporte técnico desenvolvido com Flask e SQLite.

## 📋 Funcionalidades

- ✅ Abrir chamados com título, descrição, categoria e prioridade
- ✅ Identificação do solicitante (nome, e-mail e telefone)
- ✅ Listar todos os chamados com filtros avançados
- ✅ Visualizar detalhes de cada chamado
- ✅ Atualizar chamados (editar informações e status)
- ✅ Excluir chamados
- ✅ Status dos chamados (Aberto, Em Andamento, Aguardando, Resolvido, Fechado)
- ✅ Sistema de mensagens flash para feedback ao usuário
- ✅ Design responsivo
- ✅ Estatísticas e rankings
- ✅ Pesquisa em todos os campos
- ✅ Filtros por status, prioridade e solicitante

## 🚀 Tecnologias Utilizadas

- **Backend:** Python com Flask
- **Banco de Dados:** SQLite
- **Frontend:** HTML5, CSS3, JavaScript
- **Template Engine:** Jinja2
